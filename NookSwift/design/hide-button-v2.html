方案对比 — 把 × 换成"隐藏 / 收起"语义的图标

Nook 是 menu-bar 风格的 agent app，× 按钮实际只"隐藏面板"（进程继续后台跑等热角再呼出）。但 × 在 macOS 上几乎都意味着"关闭"，让朋友以为彻底退出了 → 困惑。 下面 5 个候选，我用真实 Nook header 的样子展示，每个都有 hover 提示。
A · 当前 × — 容易被理解为"退出/关闭"，朋友测出歧义
收集箱
添加任务
钉住面板
排序
设置
关闭？退出？
点 × 后 Nook 实际还在后台跑，但用户以为已退出
B · ← 返回箭头（左滑） — 暗示"收回到原位置"
收集箱
添加任务
收回到屏幕左侧（仍在后台）
面板从左滑出 → 用 ← 暗示"滑回去"
C · ⤴ 抽屉收起（向左上） — 让用户想到"塞回去"
收集箱
收回到角落（撞角再呼出）
指向触发热角的方向
D · 折叠图标 — chevron 收起（仍是常见交互符号）
收集箱
收起面板
« 双 chevron 是经典"收起 sidebar"图标
E · ⌄ 下沉抽屉 — 短粗 chevron-down，表示"放下/收下去"
收集箱
收起面板
macOS popup / disclosure 的常见模式（最眼熟）
F · 眼睛闭上 (eye-off) — 字面"hide"
收集箱
隐藏面板
最直白，但视觉略复杂

我的推荐

方案 B（← 左箭头）：跟"面板从左滑入"的物理动画一致，朋友一眼明白"收回去"
或 方案 D（«双chevron）：sidebar 收起的国际通用图标，技术朋友更熟悉
不推荐 F：eye-off 视觉太重
不推荐 E：⌄ 是 dropdown 暗示，跟"隐藏"语义不直接
